Erster Login













Beim ersten Start der Anwendung steht noch nicht viel zur Verfügung, da kein User eingeloggt ist. Diese Hilfe kann aber jederzeit erreicht werden.

Im Auslieferungszustand gibt es einen Administrator-Account, den man für den ersten Login verwenden kann:

Username: admin
Passwort: admin

Die nächsten Schritte

Folgende Möglichkeiten stehen nun zur Auswahl:

  • BikeDB soll mehreren Anwendern zur Verfügung stehen: siehe Multi-User

  • BikeDB soll alleine genutzt werden: siehe Einzelplatz-Anwendung

  • Man legt sich als Administrator einen persönlichen Benutzer an: siehe Neuen Benutzer anlegen

Um unerlaubten Zugriff auf BikeDB zu verhindern, sollte man in jedem Falle das Passwort des Standard Administrators ändern. Siehe Sicherheitshinweis.
