Die LibUSB wird unter Windows nur benötigt, wenn man die Daten einer Sigma Docking Station (https://sigma.bike/de/) auslesen will.
Derzeit wird nur die Version TOPLINE 2009/2012 mit folgenden Fahrrad Computern unterstützt:
BC 5.09
BC 5.12 
BC 8.12 
BC 12.12 
BC 14.12 ALT 
BC 16.12 
BC 8.1 ATS 
BC 12.12. STS 
BC 16.12 STS/CAD 
BC 1909 HR 
BC 2209 
Targa Rox 5.0 
Rox 6.0

Es kann sein, dass der richtige USB-Treiber aber schon zur Verfügung steht. Die Installation sollte man also nur durchführen,
wenn die Docking Station in Kombination mit der BikeDB nicht funktioniert. BikeDB sollte aber auch ohne Installation funktionieren.


==================================================================================================================================

IMPORTANT, please read carefully!

LibUSB only needs to be installed if the Sigma Docking Station (https://sigma.bike) is to be used to transfer data from the bike
computer to BikeDB. Currently only version TOPLINE 2009/2012 is supported together with the following bike computers:
BC 5.09
BC 5.12 
BC 8.12 
BC 12.12 
BC 14.12 ALT 
BC 16.12 
BC 8.1 ATS 
BC 12.12. STS 
BC 16.12 STS/CAD 
BC 1909 HR 
BC 2209 
Targa Rox 5.0 
Rox 6.0

It is possible that the correct USB driver is already available. Thus, the installation is only needed if the docking station 
cannot be accessed by BikeDB which should normally work without the installation of LibUsbDotNet.
